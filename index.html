Sistema de gestión contable

Construido con JavaScript, PHP, HTML5, CSS, Bootstrap, jQuery y SQL es un sistema enfocado en optimizar los procesos contables de una empresa de manufactura.

  • Mirar demo

¿Qué ofrece este sistema?

El sistema funciona como una aplicación web. Optimiza el proceso de registro de ventas, gestión de clientes, rutas de ventas, productos, roles de usuario, historiales de ventas y de manipulación del sistema, generación de reportes, cuentas por pagar, etc. Permite realizar auditorías y ofrece seguridad gracias a el acceso unicamente a través de cuentas de usuario.

Escalabilidad

Todo el sistema está construido de manera que más modulos puedan ser añadidos a todo el ecosistema sin afectar a la base principal con la que funciona de manera inicial. De igual modo, gracias a la organización de toda la información los usuarios pueden generar consultas dinámicas y no limitarse unicamente a las que el sistema ofrece de manera nátiva.

Velocidad y eficiencia

Gracias al uso de tecnologías web para el desarrollo del sistema la velocidad de ejecución no es un problema, no requiere ser ejecutado en un equipo o sistema operativo especifico, basta con el uso de un navegador. Además, el navegar dentro del sistema es un ejercicio fluido y que permite realizar el trabajo de manera más eficiente y organizada.

Descripción de módulos del sistema

El sistema cuenta con trece módulos para la gestión contable de la empresa, cada móduo está interconectado con otro.

Registro de ventas principal

Es el módulo de registro de todas las ventas que se realizan, contiene campos obligatorios y validaciones por JavaScript como método de seguridad para el ingreso correcto de información.

Editar datos personales

Este módulo permite al usuario en sesión cambiar su contraseña de acceso, su foto de perfil, editar información personal de contácto como teléfonos, direcciones o información adicional que el administrador debe saber.

Gestión de usuarios

Este módulo es especifico para administradores del sistema, permite editar los roles de usuario, permisos, eliminar o dar de alta usuarios o consultar información personal de los usuarios registrados en el sistema.

Historial de movimientos

Módulo reservado para los administradores, lleva un historial de todos los movimientos que se relizan en el sistema, ejemplo; creación de una nueva ruta, registro de un cliente, etc.

Notas pendientes

Este modulo lleva un registro de todas las notas pendientes que existen en el sistema, además añade la opción para abonar a esas cuentas, liquidarlas o ver a detalle la información de cada cuenta pendiente.

Gestión de clientes

Permite la gestión de todos los clientes. Además permite ver la información de cada cliente, el promedio de compra, la deuda actual de cada cliente, así como los datos de contacto.

Reporte

Permite generar un reporte en tiempo real del registro de las ventas de cada ruta, además proporciona información desglosada de los productos y cantidades vendidas asi como de el monto pagado a traves de los diferentes métodos que la empresa acepta.

Rutas

Permite la gestión de las diversas rutas de ventas con las que la empresa cuenta, se pueden crear nuevas rutas, cambiar la imagen de cada ruta y el nombre de cada ruta.

Productos

Permite la gestión de todos los productos que la empresa manufactura, añade los detalles de cada producto y los establece como disponibles para su futura venta o registro.

Historial de ventas

Este módulo guarda toda la información de todas las ventas, abonos o liquidos que se han realizado en el sistema, guarda con detalle cada acción de este tipo y permite también ver los detalles de cada venta registrada.

Vendedores

Este módulo permite la gestión de los vendedores de la empresa, el registro de ellos así como la consulta de todos los datos de interés.

Embarques y desembarques

Permite el registro de embarques y desembarques de las unidades de venta que tiene la empresa, esto permite tener un mejor control de las entradas y salidas de productos.
